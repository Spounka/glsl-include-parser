Detailed Description

Parser is a quick and fast GLSL include parser class.

A class made specifically for GLSL language to make including other shader code easier similar to what a #include <header> in C would do.

Example:

material.frag

struct Material
{
vec3 Color;
sampler2D diffuse;
};
uniform Material material;

fragment.frag

#version 330 core
out vec4 FragColor;
in vec2 TextCoords;
#include <material.frag>
void main()
{
FragColor = vec4(material.Color, 1.0) * texture(material.diffuse, TextCoords);
}
